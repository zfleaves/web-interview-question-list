# 7. CSS 性能优化有哪些方法？

**答案：**

**优化方法：**

```css
/* 1. 使用简写属性 */
/* ❌ 不推荐 */
.box {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}

/* ✅ 推荐 */
.box {
  margin: 10px 20px;
}

/* 2. 避免过度嵌套 */
/* ❌ 不推荐 */
.container .header .nav .item .link {
  color: blue;
}

/* ✅ 推荐 */
.nav-link {
  color: blue;
}

/* 3. 使用 transform 代替 position */
/* ❌ 不推荐 */
.box {
  position: absolute;
  left: 100px;
  top: 100px;
}

/* ✅ 推荐 */
.box {
  transform: translate(100px, 100px);
}

/* 4. 使用 will-change */
.box {
  will-change: transform;
}

.box:hover {
  transform: scale(1.1);
}

/* 5. 避免使用 @import */
/* ❌ 不推荐 */
@import url('styles.css');

/* ✅ 推荐 */
<link rel="stylesheet" href="styles.css">

/* 6. 压缩 CSS */
/* 使用工具：cssnano, clean-css */
```
