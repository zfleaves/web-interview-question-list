## 13.  什么是 HTTP 代理？代理服务器的作用是什么？

**答案：**

### HTTP 代理简介

HTTP 代理服务器是位于客户端和服务器之间的中间服务器，代表客户端向服务器发送请求，并将服务器的响应返回给客户端。

### 代理服务器的作用

1. **缓存**：缓存服务器响应，减少重复请求
2. **过滤**：过滤恶意内容、广告等
3. **匿名**：隐藏客户端真实 IP 地址
4. **加速**：通过缓存和压缩加速访问
5. **监控**：记录访问日志，监控网络流量
6. **负载均衡**：分发请求到多个服务器

### 代理类型

1. **正向代理**：代表客户端访问服务器
2. **反向代理**：代表服务器接受客户端请求
3. **透明代理**：客户端不知道代理的存在
4. **匿名代理**：隐藏客户端 IP
5. **高匿代理**：隐藏代理本身的存在

### 正向代理 vs 反向代理

| 特性 | 正向代理 | 反向代理 |
|------|----------|----------|
| 代理对象 | 客户端 | 服务器 |
| 使用场景 | 公司内网、翻墙 | 负载均衡、缓存 |
| 配置位置 | 客户端 | 服务器 |
| 真实 IP | 隐藏客户端 | 隐藏服务器 |
