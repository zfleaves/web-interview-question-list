# 6. 如何设计组件库的项目结构？

**答案：**

```
my-component-library/
├── packages/
│   ├── components/      # 组件包
│   ├── theme/           # 主题包
│   ├── utils/           # 工具包
│   └── icons/           # 图标包
├── docs/                # 文档
├── scripts/             # 构建脚本
├── package.json
├── pnpm-workspace.yaml
└── tsconfig.json
```
