# 36. 如何使用结构化数据优化 SEO？

**答案：**

结构化数据可以帮助搜索引擎更好地理解页面内容。

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Article Title",
  "author": {
    "@type": "Person",
    "name": "Author Name"
  },
  "datePublished": "2025-01-08"
}
</script>
```
