# 10. 如何优化循环性能？

**答案：**

优化循环可以减少不必要的计算，提高执行效率。

```javascript
// ❌ 低效的循环
for (let i = 0; i < array.length; i++) {
  // 每次都计算 array.length
}

// ✅ 优化后的循环
const length = array.length;
for (let i = 0; i < length; i++) {
  // 只计算一次 length
}

// ✅ 使用 for...of（适用于需要遍历值的情况）
for (const item of array) {
  // 更简洁
}

// ✅ 使用 forEach（适用于简单遍历）
array.forEach(item => {
  // 更简洁
});
```
