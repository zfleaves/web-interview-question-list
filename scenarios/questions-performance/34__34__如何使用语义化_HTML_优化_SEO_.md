# 34. 如何使用语义化 HTML 优化 SEO？

**答案：**

语义化 HTML 可以帮助搜索引擎更好地理解页面内容。

```html
<header>
  <nav>
    <ul>
      <li><a href="/">Home</a></li>
      <li><a href="/about">About</a></li>
    </ul>
  </nav>
</header>

<main>
  <article>
    <h1>Article Title</h1>
    <p>Article content...</p>
  </article>
</main>

<footer>
  <p>&copy; 2025 My Website</p>
</footer>
```
