# 10. 如何优化 CSS 选择器和动画？

**答案：**

减少选择器复杂度，使用 transform 和 opacity 实现动画。

```css
/* ❌ 复杂的选择器 */
body div.container ul li a span.text { color: red; }

/* ✅ 简单的选择器 */
.text { color: red; }

/* ✅ 使用 transform 和 opacity 实现动画 */
.element {
  transform: translateX(100px);
  opacity: 0;
  transition: all 0.3s;
}
```
