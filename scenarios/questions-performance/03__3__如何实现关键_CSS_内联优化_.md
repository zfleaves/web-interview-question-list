# 3. 如何实现关键 CSS 内联优化？

**答案：**

将关键 CSS 内联到 HTML 中，减少阻塞渲染的时间，非关键 CSS 异步加载。

```html
<!-- 内联关键 CSS -->
<style>
  body { margin: 0; }
  .header { height: 60px; }
</style>

<!-- 异步加载非关键 CSS -->
<link rel="preload" href="/styles/main.css" as="style" onload="this.onload=null;this.rel='stylesheet'">
<noscript><link rel="stylesheet" href="/styles/main.css"></noscript>
```
