# 9. 如何避免内存泄漏？

**答案：**

及时清理定时器、事件监听器，使用 WeakMap/WeakSet。

```javascript
// ✅ 正确清理
function Component() {
  const timer = setInterval(() => {
    console.log('tick');
  }, 1000);

  return () => {
    clearInterval(timer);
  };
}

// ✅ 使用 WeakMap/WeakSet
const cache = new WeakMap();
cache.set(element, data);
// 当 element 被回收时，cache 中的数据也会被回收
```
