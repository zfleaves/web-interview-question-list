# 10. 判断括号是否匹配

**答案：**

```javascript
function isMatch(str) {
  const stack = [];
  const map = {
    '(': ')',
    '[': ']',
    '{': '}'
  };
  
  for (const char of str) {
    if (char in map) {
      stack.push(char);
    } else {
      const top = stack.pop();
      if (!top || map[top] !== char) {
        return false;
      }
    }
  }
  
  return stack.length === 0;
}

// 测试
console.log(isMatch('()')); // true
console.log(isMatch('()[]{}')); // true
console.log(isMatch('(]')); // false
console.log(isMatch('([)]')); // false
console.log(isMatch('{[]}')); // true
```

**时间复杂度：** O(n)
**空间复杂度：** O(n)
