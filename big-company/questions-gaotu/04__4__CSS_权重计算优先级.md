# 4. CSS 权重计算优先级

**答案：**

**CSS 权重计算优先级：**

```
!important > inline > id > class > 标签选择器 > 通配符选择器
```

**示例：**

```css
/* !important 最高优先级 */
#container {
  color: red !important;
}

/* id > class */
#header {
  color: blue;
}
.header {
  color: green; /* 被覆盖 */
}

/* class > 标签选择器 */
.container {
  color: yellow;
}
.container {
  color: purple;
}

/* 通配符选择器 */
div {
  color: orange;
}
.container div {
  color: pink;
}

/* 继承 */
.container {
  color: gray;
}
.container div {
  color: brown;
}
```
