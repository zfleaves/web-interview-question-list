# 44. Vue3 的响应式系统处理数组有哪些注意事项？

**答案：**

```javascript
// ❌ 不要直接替换整个数组
state.items = [1, 2, 3];  // 会失去响应式

// ✅ 使用 push/splice 等方法
state.items.push(1, 2, 3);

// ✅ 或者重新赋值
state.items = reactive([1, 2, 3]);
```
