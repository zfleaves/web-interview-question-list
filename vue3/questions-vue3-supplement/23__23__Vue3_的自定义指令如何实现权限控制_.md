# 23. Vue3 的自定义指令如何实现权限控制？

**答案：**

```javascript
app.directive('permission', {
  mounted(el, binding) {
    const { value } = binding;
    const permissions = ['admin', 'editor'];

    if (!permissions.includes(value)) {
      el.parentNode?.removeChild(el);
    }
  }
});

// 使用
<button v-permission="'admin'">Admin Only</button>
```
