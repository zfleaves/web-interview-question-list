## 1. 小程序和 H5 的区别是什么？

**答案：**

### 运行环境

| 特性 | 小程序 | H5 |
|------|--------|-----|
| 运行环境 | 微信提供的宿主环境 | 浏览器环境 |
| 渲染引擎 | WebView + 原生组件 | WebView |
| JS 引擎 | V8 | V8 |
| API 能力 | 微信提供的 API | 标准 Web API |

### 开发差异

```javascript
// 小程序
Page({
  data: {
    message: 'Hello'
  },
  onLoad() {
    console.log('页面加载');
  }
});

// H5
class App extends React.Component {
  state = {
    message: 'Hello'
  };

  componentDidMount() {
    console.log('组件挂载');
  }

  render() {
    return <div>{this.state.message}</div>;
  }
}
```

### 性能差异

- **小程序**：加载更快，体验更接近原生
- **H5**：加载较慢，但跨平台能力强
