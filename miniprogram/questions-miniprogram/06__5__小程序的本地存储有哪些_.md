## 5. 小程序的本地存储有哪些？

**答案：**

### wx.setStorage / wx.getStorage

```javascript
// 异步存储
wx.setStorage({
  key: 'token',
  data: 'abc123',
  success() {
    console.log('存储成功');
  }
});

// 异步读取
wx.getStorage({
  key: 'token',
  success(res) {
    console.log(res.data); // abc123
  }
});

// 同步存储
try {
  wx.setStorageSync('token', 'abc123');
} catch (e) {
  console.error(e);
}

// 同步读取
try {
  const token = wx.getStorageSync('token');
  console.log(token); // abc123
} catch (e) {
  console.error(e);
}
```

### wx.removeStorage / wx.clearStorage

```javascript
// 删除指定 key
wx.removeStorage({
  key: 'token',
  success() {
    console.log('删除成功');
  }
});

// 清空所有数据
wx.clearStorage({
  success() {
    console.log('清空成功');
  }
});
```

### 存储限制

- 单个 key 允许存储的最大数据长度为 1MB
- 所有数据存储上限为 10MB
