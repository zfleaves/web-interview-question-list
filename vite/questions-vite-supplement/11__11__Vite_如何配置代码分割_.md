# 11. Vite 如何配置代码分割？

**答案：**

```javascript
// vite.config.js
export default {
  build: {
    rollupOptions: {
      output: {
        manualChunks: {
          'vue-vendor': ['vue', 'vue-router', 'pinia'],
          'utils': ['lodash', 'axios']
        }
      }
    }
  }
};

// 动态导入
const LazyComponent = defineAsyncComponent(() => import('./LazyComponent.vue'));
```
