# 12. Vite 如何配置代理解决跨域？

**答案：**

```javascript
// vite.config.js
export default {
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true,
        rewrite: (path) => path.replace(/^\/api/, '')
      },
      '/files': {
        target: 'http://localhost:3000',
        changeOrigin: true
      }
    }
  }
};
```
